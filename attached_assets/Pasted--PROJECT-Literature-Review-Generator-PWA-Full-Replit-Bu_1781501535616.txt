# PROJECT: Literature Review Generator (PWA) — Full Replit Build Prompt

Build a **production-ready Progressive Web App (PWA)** called:

> **Literature Review Generator**

A free, AI-powered academic research assistant for generating literature reviews, analyzing papers, detecting research gaps, and managing research workflows.

The app must use:

* Python Flask
* SQLite database
* Free academic APIs
* Groq AI (primary)
* PWA support (installable + offline mode)

---

# CORE REQUIREMENTS

## PUBLIC ACCESS SYSTEM (NO LOGIN)

DO NOT implement:

* User registration
* User login
* Authentication system
* Passwords
* User profiles
* Roles or permissions
* Subscription or payment system

All features must be publicly accessible to anyone using the app.

## 1. Paper Discovery System

Integrate free academic APIs:

* OpenAlex
* Semantic Scholar
* Crossref
* PubMed
* arXiv

Users can:

* Search academic papers
